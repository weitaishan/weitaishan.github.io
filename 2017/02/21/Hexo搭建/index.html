前言

很早之前就想搭建一个自己的博客了，记录一些自己的所获所得，一些乱七八糟的事都想记录下来，由于种种原因(ps:其实就是懒)也没有将自己的想法付诸于实践，时间就总是在一些琐碎的事之中慢慢流逝，而我还是一无所获。

然后又不知道什么时候冒出的念头，连一丝犹豫都没有，就去GoDaddy买了域名，又问了一下大神用的什么搭建博客，最后觉得Hexo名字很好听，又符合我简约的审美观，就选择了Hexo搭建博客，开始觉得很困难，但是当你认真去做一件事的时候，才发现有些东西并没有你想象之中那么困难，So，有想法的孩子赶紧一起来搭建一个属于自己的博客吧，你想怎么蹂躏就怎么蹂躏。
